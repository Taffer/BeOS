DragonDrop — a drag 'n' drop demo

DragonDrop is a little demo application demonstrating the concepts documented in the BeBook's drag-n-drop documentation. It also does several other interesting (to developers) things:

  • all strings are stored in resources, allowing for easy translation (loaded via BResourceStrings)
  • loads bitmaps from the resources (using the BTranslationUtils class)
  • accepts any kind of bitmap from drag-n-drop (courtesy of the Translation Kit)
  • can send any kind of bitmap during a drag-n-drop (courtesy of the Translation Kit)
  • can create clipping files in any kind of bitmap format by dragging to the Tracker or a Tracker-like application (courtesy of the Translation Kit)

The code is also extensively documented; it should be easy to follow.

About the Images

The spiffy images that are built into this application's resources are PNG images, and thus have their own built-in alpha channel, which makes them interesting for drag-n-drop operations.
